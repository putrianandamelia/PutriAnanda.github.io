Pengantar Generic

Generic adalah fitur dalam pemrograman yang memungkinkan suatu kode digunakan untuk berbagai tipe data tanpa harus menuliskan ulang kode yang sama.

  • Tidak perlu membuat kode berulang.
  • Tipe data lebih terkontrol.
  • Error bisa dicegah lebih awal.

Masalah Tanpa Generic

Pada slide ini digunakan dynamic, sehingga data bisa berisi tipe apa saja tanpa pengecekan.

✔ Success

  • Data yang disimpan memang bertipe int

  dynamic data;

  Box(this.data);
}

void main() {
  var box = Box(42);

// Tidak aman!

  print(value);
}

  • Data yang disimpan tidak sesuai dengan tipe yang digunakan
  • Data ternyata String, tetapi dipaksa jadi int

Sintaks Generasi Dasar

Sintaks generic digunakan dengan tanda < > untuk menentukan parameter tipe data.

  • T : Type (umum).
  • E : Element.
  • K, V : Key dan Value.
  • R : Return Type.

Contoh Sintaks Generic pada class dan function

Pada slide ini dijelaskan penggunaan generic pada class dan function, yaitu dengan menambahkan sebagai parameter tipe agar bisa digunakan untuk berbagai jenis data.

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class MyClass {
  T data;

  MyClass(this.data);

  T getData() {
    return data;
  }
}

T myFunc(T value) {
  return value;
}

void main() {
  // Class generic
  var myIntClass = MyClass(10);
  var myStringClass = MyClass('test');

  print(myIntClass.getData());
  print(myStringClass.getData());

  // Fungsi generic
  var intValue = myFunc(100);
  var stringValue = myFunc('hello');

  print(intValue);
  print(stringValue);
}

Pembatasan Tipe dan manfaatnya

Pada slide ini dijelaskan tentang manfaat penggunaan pembatasan tipe (constraints) pada generic. Dengan pembatasan tipe, kita bisa memastikan bahwa hanya tipe data tertentu yang dapat digunakan, sehingga kode menjadi lebih aman dan terstruktur.

  • Meningkatkan keamanan tipe data (mengurangi risiko error).
  • Membuat kode lebih mudah dipahami.
  • Memungkinkan penggunaan method tertentu dari tipe tersebut.
  • Mencegah penggunaan tipe yang tidak sesuai.
  • Membantu dalam perancangan kode yang lebih rapi.

Praktik 1 : Generic Class & Fungsi

  • Membuat class Repository(T) dengan method: save(T item), getById(int id), getAll()
  • Membuat fungsi generic swap(T) untuk menukar posisi data dalam list
  • Membuat class Triple(A, B, C) untuk menyimpan tiga tipe data berbeda

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class Repository {
  Map data = {};

  void save(int id, T item) {
    data[id] = item;
  }

  T? getById(int id) {
    return data[id];
  }

  List getAll() {
    return data.values.toList();
  }
}

void swap(List list, int i, int j) {
  var temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

class Triple {
  A first;
  B second;
  C third;

  Triple(this.first, this.second, this.third);
}

void main() {
  var repo = Repository();

  repo.save(1, "Data 1");
  repo.save(2, "Data 2");

  print(repo.getById(1));
  print(repo.getAll());

  var list = [1, 2, 3];
  swap(list, 0, 2);
  print(list);

  var t = Triple(1, "Hello", 3.14);
  print("${t.first}, ${t.second}, ${t.third}");
}

Generic vs Dynamic vs Object

1. Generic :

  • Dicek saat compile-time
  • Paling aman
  • Tidak perlu casting
2. Dyanamic :
  • Tidak dicek di awal
  • Sangat fleksibel
  • Rawan error saat runtime
3. Object :
  • Dicek saat compile-time
  • Lebih aman dari dynamic
  • Tetapi perlu casting untuk digunakan

// Contoh perbedaan
Box box1 = Box(42); // Generic: aman
dynamic box2 = 42; // Dynamic: tidak aman
Object box3 = 42; // Object: perlu casting

Ringkasan

Latihan

Latihan pada slide ini bertujuan untuk melatih pemahaman tentang penggunaan generic dalam pembuatan class dan function.

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  

class Stack {
  List data = [];

  void push(T nilai) {
    data.add(nilai);
  }

  T? pop() {
    if (data.isEmpty) return null;
    return data.removeLast();
  }

  T? peek() {
    if (data.isEmpty) return null;
    return data.last;
  }
}

List mergeLists(List a, List b) {
  List hasil = [];
  hasil.addAll(a);
  hasil.addAll(b);
  return hasil;
}

class Response {
  T? data;
  String? error;

  Response({this.data, this.error});
}

void main() {
  // stack
  var s = Stack();
  s.push(1);
  s.push(2);
  print(s.peek());
  print(s.pop());

  // merge list
  var gabung = mergeLists([1, 2], [3, 4]);
  print(gabung);

  // response
  var r = Response(data: "sukses");
  print(r.data);
}
