Penjelasan Polimorfisme dan Abstraksi

Polimorfisme adalah konsep dalam pemrograman berorientasi objek di mana satu tipe atau satu method dapat memiliki banyak bentuk atau perilaku. Artinya, objek yang berbeda bisa merespons method yang sama dengan cara yang berbeda, tergantung dari class masing-masing. Abstraksi adalah konsep menyederhanakan sistem dengan menyembunyikan detail implementasi dan hanya menampilkan bagian pentingnya saja. Dengan abstraksi, kita fokus pada apa yang dilakukan suatu objek, bukan bagaimana cara kerjanya.

Pewarisan

Inheritance adalah konsep dimana class dapat mewarisi sifat dan method dari class lain menggunakan extends. Method dari parent bisa digunakan kembali atau diubah dengan @override, dan constructor parent dapat dipanggil menggunakan super(). 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

  void bersuara() => print('...');
}

class Sapi extends Hewan {
  @override
  void bersuara() => print('Moo');
}

void main() {
  Sapi s = Sapi();
  s.bersuara();
}

Pengantar Polimorfisme

Polimorfisme berarti satu objek memiliki banyak bentuk. Objek dari subclass bisa diperlakukan sebagai superclass, tetapi tetap menjalankan perilaku sesuai dengan class aslinya. Dalam Dart, polimorfisme terjadi saat runtime melalui method overriding.

Satu variabel parent bisa menyimpan berbagai object child.

Pengantar Abstraksi

Abstraksi adalah konsep menyembunyikan detail implementasi dan hanya menampilkan fungsi utama. Tujuannya agar fokus pada apa yang dilakukan sistem, bukan bagaimana cara kerjanya

Abstract Class

Abstract class adalah class yang tidak bisa diinstansiasi secara langsung. Class ini dapat memiliki method abstract (tanpa isi) dan method biasa (dengan isi) sebagai dasar bagi subclass. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

abstract class Kendaraan {
  String? merk;

  void start();

  void stop() {
    print('Kendaraan berhenti');
  }
}

class Motor extends Kendaraan {
  @override
  void start() {
    print('Motor $merk menyala');
  }
}

void main() {
  Motor m = Motor();
  m.merk = "Yamaha";
  m.start();
  m.stop();
}

Abstract Method

Abstract method adalah method tanpa isi yang wajib diimplementasikan oleh subclass. Hal ini memastikan setiap subclass memiliki fungsi tertentu sesuai aturan.

class Biola extends AlatMusik {

  void mainkan() {
    print('Ngeeeng... ngeeeng...');
  }
}

void main() {

Interface di Dart

Interface adalah kontrak yang harus diikuti oleh class. Dalam Dart, interface digunakan dengan keyword implements, dan semua method harus diimplementasikan. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
abstract class Flyable {
  void terbang();
}

class Roket implements Flyable {
  @override
  void terbang() => print('Roket meluncur ke angkasa');
}

class BalonUdara implements Flyable {
  @override
  void terbang() => print('Balon udara melayang di langit');
}

void main() {
  Roket b = Roket();
  BalonUdara p = BalonUdara();

  b.terbang();
  p.terbang();
}

Perbedaan extends vs implements

extends digunakan untuk pewarisan (is-a), sedangkan implements digunakan untuk menerapkan kontrak (can-do). extends hanya satu parent, sementara implements bisa lebih dari satu

Static Members dan Method

Static adalah property atau method yang dimiliki oleh class, bukan objek. Static dapat diakses langsung tanpa membuat instance, biasanya digunakan untuk fungsi umum

  print('Result: ${MathUtils.add(12, 18)}');

Studi Kasus: Sistem Bentuk Geometri

Class parent digunakan sebagai dasar untuk berbagai bentuk seperti segitiga dan persegi panjang. Setiap bentuk memiliki cara perhitungan luas dan keliling yang berbeda. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
abstract class Bentuk {
  double hitungLuas();
  double hitungKeliling();

  void info() {
    print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}');
  }
}

class Segitiga extends Bentuk {
  double alas, tinggi, sisiMiring;

  Segitiga(this.alas, this.tinggi, this.sisiMiring);

  @override
  double hitungLuas() => 0.5 * alas * tinggi;

  @override
  double hitungKeliling() => alas + tinggi + sisiMiring;
}

class PersegiPanjang extends Bentuk {
  double panjang, lebar;

  PersegiPanjang(this.panjang, this.lebar);

  @override
  double hitungLuas() => panjang * lebar;

  @override
  double hitungKeliling() => 2 * (panjang + lebar);
}

void main() {
  Bentuk b1 = Segitiga(4, 6, 5);
  Bentuk b2 = PersegiPanjang(5, 3);

  b1.info();
  b2.info();
}

Kesimpulan

polimorfisme dan abstraksi adalah konsep penting dalam pemrograman berorientasi objek yang membantu membuat program lebih terstruktur dan efisien. Polimorfisme memungkinkan satu bentuk memiliki banyak perilaku sehingga program menjadi lebih fleksibel, sedangkan abstraksi menyederhanakan sistem dengan menyembunyikan detail yang tidak perlu dan menampilkan fungsi utama saja. Keduanya membuat kode lebih mudah dikembangkan, digunakan kembali, dan dirawat.

Bonus 3: Class Mathutils

Membuat abstract class sebagai kerangka dasar, lalu diimplementasikan oleh class turunan. Setiap subclass wajib mengisi method yang sudah ditentukan sehingga program lebih terstruktur. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
import 'dart:math';

class MathUtils {
  static int faktorial(int n) {
    if (n <= 1) return 1;
    return n * faktorial(n - 1);
  }

  static bool isPrima(int n) {
    if (n <= 1) return false;
    for (int i = 2; i <= sqrt(n); i++) {
      if (n % i == 0) return false;
    }
    return true;
  }

  static double roundDouble(double value, int places) {
    double mod = pow(10, places).toDouble();
    return (value * mod).round() / mod;
  }

  static int randomInt(int min, int max) {
    return min + Random().nextInt(max - min + 1);
  }

  static double randomDouble(double min, double max) {
    return min + Random().nextDouble() * (max - min);
  }

  static double rataRata(List data) {
    return data.reduce((a, b) => a + b) / data.length;
  }

  static double median(List data) {
    data.sort();
    int n = data.length;

    if (n % 2 == 1) {
      return data[n ~/ 2];
    } else {
      return (data[n ~/ 2 - 1] + data[n ~/ 2]) / 2;
    }
  }
}

void main() {
  print(MathUtils.faktorial(5));
  print(MathUtils.isPrima(7));
  print(MathUtils.roundDouble(3.14159, 2));
  print(MathUtils.randomInt(1, 10));
  print(MathUtils.randomDouble(1, 5));
  print(MathUtils.rataRata([10, 20, 30]));
  print(MathUtils.median([10, 20, 30, 40]));
}
