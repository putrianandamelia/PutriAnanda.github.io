Soal 1: Konflik Nama Antar Extension

  • Jawaban:

    • Kode mengalami error karena terdapat dua extension pada tipe String yang memiliki nama method sama yaitu toInt(). Saat angka.toInt() dipanggil, Dart tidak tahu harus menggunakan method dari extension StringNumber atau StringDouble. Dart juga tidak membedakan method berdasarkan return type, sehingga terjadi konflik method. Solusinya adalah menggunakan extension override agar jelas extension mana yang dipakai.

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

extension StringNumber on String {
  int toInt() {
    return int.parse(this);
  }
}

extension StringDouble on String {
  double toInt() {
    return double.parse(this);
  }
}

void main() {
  String angka = '123';

  print(StringNumber(angka).toInt());
  print(StringDouble(angka).toInt());
}

Soal 2: Extension Method Tidak Bisa Mengakses Member Private

  • Jawaban:

    • Kode gagal dikompilasi karena _value adalah variabel private. Di Dart, member yang diawali _ hanya dapat diakses di dalam class/library yang sama. Extension tidak bisa langsung mengakses member private tersebut. Solusinya adalah membuat getter publik untuk mengambil nilai _value tanpa mengubahnya menjadi public.

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class Counter {
  int _value = 0;

  void increment() {
    _value++;
  }

  int get value => _value;
}

extension CounterStats on Counter {
  int getValue() {
    return value;
  }
}

void main() {
  var c = Counter();

  c.increment();

  print(c.getValue());
}
