Redirecting Constructors

Ringkasan

  • Redirecting constructor menggunakan sintaks : this(...) tanpa body.
  • Digunakan untuk memanggil constructor lain dalam class yang sama.
  • Mengurangi duplikasi kode.
  • Tidak dapat berisi logika yang kompleks

Latihan

Program ini membuat class Circle dengan constructor utama yang menerima nilai radius. Selain itu, terdapat redirecting constructor dari diameter, keliling, dan luas yang mengarahkan ke constructor utama menggunakan : this(...), sehingga kode menjadi lebih ringkas dan tidak terjadi duplikasi kode.

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

import 'dart:math';

class Circle {
  final double radius;

  Circle(this.radius);

  Circle.fromDiameter(double diameter)
      : this(diameter / 2);

  Circle.fromCircumference(double keliling)
      : this(keliling / (2 * pi));

  Circle.fromArea(double luas)
      : this(sqrt(luas / pi));
}

void main() {
  var c1 = Circle(7);
  var c2 = Circle.fromDiameter(14);
  var c3 = Circle.fromCircumference(44);
  var c4 = Circle.fromArea(154);

  print("Radius c1 = ${c1.radius}");
  print("Radius c2 = ${c2.radius}");
  print("Radius c3 = ${c3.radius}");
  print("Radius c4 = ${c4.radius}");
}
